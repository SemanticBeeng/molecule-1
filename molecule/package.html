  8. final def bridge[A](ichan: IChan[A], ochan: OChan[A]): Unit

    Bridge a system-level input interface to a system-level output interface.

    Bridge a system-level input interface to a system-level output interface. All the segments coming on the input are forwarded to the output in a tight immutable loop in the threads of the sender and the receiver without further level of indirection. If the input terminates first, the output is closed with the termination signal of the input. If the output terminates first, the input is poisoned with the termination signal of the output.

    ichan

    the input.

    ochan

    the output.
